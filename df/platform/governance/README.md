# Governance Platform

Governance turns datasets into enterprise data products that can be discovered, trusted and audited.

## P0 Capabilities

- Data product ownership and lifecycle status.
- Catalog registration metadata.
- Column-level PII and sensitivity classification.
- Lineage requirements from source event to Gold consumer.
- Retention and erasure policy.
- Glossary alignment for enterprise concepts such as person, organization, product, activity,
  outcome, benefit, payment, recommendation and compliance evidence.
- Catalog bundle export for local DataHub/OpenMetadata handoff evidence.
