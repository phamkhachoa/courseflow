# Enterprise Data Platform Product

`enterprise-data-platform` is the internal platform product that owns cross-product control-plane
data products such as Data Product Control Tower outputs. It is not an LMS product; it represents the
group-wide data platform operated for all enterprise products.

Initial scope:

- Data product inventory and ownership.
- Contract, quality, lineage and release evidence aggregation.
- Governed enterprise KPI and executive scorecard materialization from semantic metric snapshots.
- Platform maturity and production-readiness blockers.
- Enterprise reporting and governance consumers.
